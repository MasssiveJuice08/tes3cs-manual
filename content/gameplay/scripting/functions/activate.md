This function tells the object to do its default activation.

| **Object Type** | **Activation** |
| --- | --- |
| NPC | Dialogue |
| Container | Opens |
| Door | Opens |
| Weapon, armor, misc, etc | Picks Up |
| Book/Scroll | Reads |
