This package tells the actor to activate the specified ObjectID. A powerful and admittedly underutilized and undertested package.

| **Object Type**          | **Activation** |
| ------------------------ | -------------- |
| NPC                      | Dialogue       |
| Container                | Opens          |
| Door                     | Opens          |
| Weapon, armor, misc, etc | Picks Up       |
| Book/Scroll              | Reads          |
